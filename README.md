# summary

This repository holds the data and the R codes for the publication "Kann die Zahl cannabisbezogener Diagnosen mithilfe des THC-Gehalts vorhergesagt werden?"

##  prepare data.R

Loads the various data sets stored in the "input" folder, prepares the variables for analyses and stores the it to a joint data set in the "data" folder ("data.csv").

##  analyses.R

Loads the prepared data set from the "data" folder ("data.csv") and performs all analyses reported in the publication, including tables and figures.
